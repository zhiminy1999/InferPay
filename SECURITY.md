# Security Policy

## Supported Versions

We actively maintain and support the security of the latest release version of InferPay:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of our stablecoin treasury infrastructure very seriously. If you find a vulnerability, please report it privately:

- **Email**: Email us at `security@inferpay.space` with details of the vulnerability.
- **Details to Include**: Provide a description of the issue, proof-of-concept steps, and potential impacts.
- **Response Time**: We will acknowledge receipt of your report within 24 hours and provide weekly updates until the issue is fixed.

Please do not open GitHub issues for security vulnerabilities.
